Star Wars API

"Iremos trabalhar com a API da franquia de Star Wars, o SWAPI. Com ela, veremos como manipular e dispor informações de uma RESTFul API (O que é?)."

O que está posto entre aspas foi escrito por Vanessa. O arquivo HTML sofreu alguns acréscimos por minha parte, CSS foi feito inteiramente por ela, e o JavaScript era a tarefa ser cumprida. Como usuário, o que é necessário saber é que conforme os números que você repassar ao Prompt (a janelinha que aparece toda vez que você carrega esta página), será tratado e corresponderá a uma resposta da API. O código estará disponível no Repositório e no GitHub.
